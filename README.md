# Dust
Dust는 Game Maker Studio 2를 사용해 제작한 2D TopView RPG 입니다. 주인공은 덮쳐오는 미세먼지를 피해 변종 몬스터들을 처치하고, 장비를 제작하며 생존해야 합니다.
